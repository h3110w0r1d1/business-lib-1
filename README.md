# business-lib-1
Note: there are 2 of the same versions. A typescript(always up to date) and a javascript version. Go to /typescript/ for typescript and /code/ for javascript.


## How to Use
1. remove the code from the /code/ folder.
2. You now have the Library.
### Guide
**main**: examples of how to use(so far)
**business**: the main library logic
**config**: change the values in here to change specific attributes about the library. Refer to the wiki for more information about specific items. **Business.json** is the main logic file for asset bases *only in v 0.0.2a*

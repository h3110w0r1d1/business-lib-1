# Security Policy

## Supported Versions
All versions are supported, however note that the versions with an :x: are out of date or are in development.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.1a   | :white_check_mark: |
| 0.0.1b,0.0.1c   | :x: |


## Reporting a Vulnerability

reach out to the repository's owner for critical issues. Otherwise create an issue.
